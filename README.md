# 🌦️ Weather App (ReactJS)

A modern and responsive weather web application built using ReactJS and Open-Meteo API.  
It provides real-time weather updates, hourly forecasts, and historical data analysis with interactive charts.

---

## 🚀 Features

### 📍 Current Weather
- Auto-detect user location (GPS)
- Displays:
  - Temperature (Current, Min, Max)
  - Wind Speed
  - Humidity
  - Precipitation

### 📊 Hourly Forecast
- Interactive charts for:
  - Temperature (°C ↔ °F toggle)
  - Humidity
  - Precipitation

### 📅 Historical Data
- Select custom date range (up to 2 years)
- Shows:
  - Max & Min Temperature
  - Precipitation
  - Wind Speed
- Includes zoom & scroll charts

### 🎨 UI Features
- Dynamic background based on weather 🌄
- Clean card-based layout
- Fully responsive design 📱
- Smooth user experience

---

## 🛠️ Tech Stack

- ReactJS
- Open-Meteo API
- Recharts (Charts)
- ECharts (Advanced charts)
- CSS (Responsive UI)

---

## ⚡ Performance Optimization

- Default location for fast initial load
- API data caching (localStorage)

- Lightweight UI rendering
